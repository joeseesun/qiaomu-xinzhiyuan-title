# Code of Conduct

## 中文

这个项目欢迎围绕标题改写、事实边界、Agent Skill 包装方式的具体讨论。请保持直接、清楚、尊重事实，不做人身攻击，不提交未经授权的私有语料或账号信息。

我们鼓励：

- 用最小例子说明问题。
- 区分“风格偏好”和“事实错误”。
- 对夸张标题的边界提出具体修改建议。
- 尊重原始内容版权和用户隐私。

我们不接受：

- 骚扰、歧视、威胁或人身攻击。
- 泄露 API key、cookie、账号截图、私有工作簿或未授权全文。
- 用虚假事实、伪造榜单或未经证实的权威背书改写标题。

维护者可以编辑、关闭或删除违反这些边界的 issue、PR 或评论。

## English

This project welcomes concrete discussion about headline rewriting, fact boundaries, and Agent Skill packaging. Please be direct, clear, and respectful. Do not harass others, submit private corpora, expose credentials, or encourage unsupported factual claims.

Maintainers may edit, close, or remove issues, pull requests, or comments that violate these boundaries.
