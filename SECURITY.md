# Security Policy

## 中文

这个仓库是一个本地 Agent Skill，不需要账号、API key 或网络权限。正常使用时，它只读取仓库内的规则文件，并根据用户提供的文章或标题生成候选标题。

请不要在 issue、PR、截图或样例里提交：

- 未公开的文章全文
- 私有工作簿或训练数据
- API key、cookie、token、密码
- 账号后台截图或商业敏感材料

如果发现安全问题，请通过 GitHub 私信/issue 先报告最小复现，不要公开敏感数据。

## English

This repository is a local Agent Skill. It does not require accounts, API keys, or network access during normal use.

Please do not submit private article corpora, API keys, cookies, tokens, passwords, account screenshots, or confidential source material in issues or pull requests.
